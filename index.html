Último trabajo para el temario Introducción al Desarrollo Web. A continuación se hace una breve descripción de la forma de trabajar el proyecto.
Los objetivos son crear una Gitpage contenedora de un Portafolio que debe mostrar las imagenes de los proyectos realizado hasta este punto y vincularlos con las páginas de cada uno y sus respectivos códigos.

Se clonó para generar un proyecto en local y trabajar en él. Se usan clases y componentes Bootstrap. También se recurre a Fontawesome para las figuritas presentes en ella (en el footer). Se usa Grid, Card, Modal, clases Bootstrap, etcétera.

Se usan algunos elementos CSS específicos para dar el efecto de papiro o rollo (scroll) con una imagen de fondo global y otra particular en la sección Mis Trabajos (parallax).

Mis Trabajos

Curriculo Básico

Elaboración de un Curriculo básico, con imagen y una breve información personal.

Landing

Primer trabajo en donde se implementa CSS para ordenar los elementos visibles de la página. Se modifican los estilos y tamaños de las fuentes. Texto junto a las imagenes y un pie de página sencillo.

Iguanas

Manejo de otros elementos CSS, para colocar imagenes en la misma fila, y textos al lado de una imagen. También se usan estilo CSS para centrar el texto del título y dar color al fondo del body. Se incluyen tres iconos con fontawesome.

Cuppon

Trabajo en el que se implementa Bootstrap. En particular el manejo de clases como container, container-fluid, nav-bar y otras. Se incluyen componentes como Card y Form. También se usa Fontawesome para la inclusión de iconos.

Fork a otro trabajo

Trabajo consistente en hacer un Fork a un proyecto de otra persona. Modificarlo, reemplazando la información dada, por datos personales. Se debió agregar un estilo personalizado.

Gitpages

Este trabajo consistía en crear una Gitpage en base a un proyecto anterior. En este caso se usó el proyecto1, contenedor del curriculo básico.

Prueba final del primer módulo

Trabajo consistente en elaborar una gitpage directamente en el sitio. Luego clonarlo y desarrollar un Portafolio como página principal. Aquí se hace uso de Bootstrap para crear tarjetas, carrusel y modal. También se usa el efecto parallax. No contiene el botón Ver sitio, porque el resultado ya lo tiene ante sus ojos.

Fork a Proyectos de Compañeras y Compañeros del curso

Fork al Proyecto 4 de las compañeras Jennifer, Vania y compañero Felipe. En el primero se agrega un área para mensajes y se separa el botón Enviar de los elementos anteriores. En el segundo, se aumentó la separación horizontal de los elementos Card. Para Felipe solo hice Fork. Sólo se incluyen los botones para examinar los códigos.

Mensajería

Trabajo en que se hace uso de Flexbox para generar el efecto de dos secciones adyacentes: la primera con usuarios y la segunda con una imagen centrada.

E-Commerce

Trabajo en el que se implementa CSS Grid para generar las secciones que se aprecian. Una de las cuales (la principal) contiene una galería con tarjetas que incluyen imagenes y textos.

Media Queries

Trabajo en el que se implementa Media Queries para generara tres vistas en la página. Se incluye CSS Grid como soporte de la página y Flex en cada una de sus cuatro elementos componentes.

Prueba CSS Avanzado

Trabajo que usa como base el anterior: incluye el uso de Flex, CSS Grid, Media Queries y Animaciones CSS para dar distintas vistas y efectos al logo del navegador, efectos de transición en las imágenes y otros elementos.
